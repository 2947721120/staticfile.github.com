### Staticfile 网站文件

如果你觉得网站的体验可以更好，那么你可以通过给你个 repo 添加代码来改进她。文件结构：

```bash
.
├── CNAME
├── _config.yml
├── about.html                          # 简介 http://staticfile.org/about.html
├── favicon.ico
├── index.html                          # 首页 http://staticfile.org/
└── static
    ├── app.css                         # 所有页面的样式
    ├── app.js                          # 所有页面用到的 js 效果，我们使用 angular.js
    ├── logos
    └── qiniu.png
```
